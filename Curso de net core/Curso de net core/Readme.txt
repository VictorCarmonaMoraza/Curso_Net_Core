## 17 - Bucle ciclo For


Estructura de un ciclo For

 for (int i = 0; i < length; i++)
            {

            }

//Imprimos del 0 hasta el 9 mediante el ciclo For
for (int i = 0; i < 10; i++)
            {
                Console.WriteLine("Resultado {0}", i); 
            }

//Imprimimos del 10 hasta el 1 mediante el ciclo For
for (int i = 10; i > 0; i--)
            {
                Console.WriteLine("Resultado {0}", i);
            }
